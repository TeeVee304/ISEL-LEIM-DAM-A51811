# API Usage

API: The Cat API
Base URL: `https://api.thecatapi.com/v1/`

## [cite_start]API endpoint [cite: 131]
`GET https://api.thecatapi.com/v1/images/search?limit=10`

## Response format
JSON Array of Objects
