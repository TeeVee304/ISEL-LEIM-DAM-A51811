# Overview

**Purpose of the application**
The "Cat Image Database" is a mobile application designed to retrieve and display random cat images from a public API. 

**Target users**
Cat enthusiasts, pet lovers, and anyone looking for a quick, entertaining feed of feline photography.

**General idea of how the system works**
The Android application uses Kotlin and XML Views to display a scrollable interface of images. It connects to the external `https://thecatapi.com` service to retrieve photo data. Users can browse the images and use a refresh action to request a new batch of cats from the network.
